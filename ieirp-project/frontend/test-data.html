📊 What to Test

Frontend Functionality

  • ✅ Navigation between pages works
  • ✅ Forms display correctly
  • ✅ Responsive design (resize browser)
  • ✅ Error messages show properly
  • ✅ Loading states work

Backend Functionality (when running)

  • ✅ API endpoints respond correctly
  • ✅ Authentication works
  • ✅ Database operations work
  • ✅ Role-based access control works
  • ✅ CORS configuration works

🔧 Setup Instructions

For Full Backend Testing:

  1. Install Java 17+: Download from adoptium.net
  2. Install MySQL 8.0+: Download from dev.mysql.com
  3. Create Database: CREATE DATABASE ieirp_db;
  4. Start Backend: cd backend && mvn spring-boot:run
  5. Access Full Application: Frontend at port 8081, Backend at port 8080

🐛 Troubleshooting

Common Issues:

  • "Failed to load categories" error: Backend not running on port 8080
  • Login fails: Check credentials and backend connectivity
  • CORS errors: Backend CORS configuration issue
  • Database errors: MySQL not running or database doesn't exist
